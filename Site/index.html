The DnD Assistant of your Dreams

DMBot helps you manage your online Dungeons and Dragons games in the pandemic. It allows users to create and save characters on the platform, rolls dice, manages combat, and has a super-useful monster lookup!

Getting Started

First, you'll need to add the bot to your Discord server. You can do so here. Next, you can create a character using the $character create command. DMBot will then message you and guide you through the character creation process. After you get a group of friends together, you can use $session start and $session end to start and end sessions.

After starting a session, you can add your character to the session by using $session add your-character-name. Now, all that's left is to have fun. Keep in mind DMBot can do other useful things like manage combat and roll dice for you - check those features out on the documentation page.
