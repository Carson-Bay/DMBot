Code music. Live. With friends.

PiBot integrates the SonicPi program to let users code music in a Discord chat room with their friends. The pandemic makes things hard, especially for those who create music. Thus, our team created PiBot for SacHacks 2021.

Getting Started

First, you'll need to add the bot to your Discord server. You can do so here. As for everything else... I don't know. We'll see.
