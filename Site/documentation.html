Commands

Character Management

  • $character create - The bot will DM the sending user and guide them through character creation.
  • $character list - The bot will DM the user a list of their characters.
  • $character show [name] - Shows the specified character sheet.
  • $character add [name] <amount=1> - Adds amount of an item/spell to the character's inventory.
  • $character delete [name] - Deletes the specified character. WARNING: This cannot be undone.
  • $character revive [name] - Revives the specified (dead) character.

Session Management

  • $session start - Starts a play session.
  • $session add [name] - Adds the specified character to the current session.
  • $session list - Lists all characters in the session.
  • $session end - Ends the current session and removes all characters.

Combat Management

  • $combat start - Starts a combat encounter, rolls initiative for all characters in the session, and displays combat turn order.
  • $combat damage [name] - Damages the specified character in the session. If that character's HP is zero, the bot notifies that they are dead/knocked out.
  • $combat end - Ends the current combat encounter.

Utility Commands

  • $roll <amount=1> d[sides] - Rolls amount of dice with the specified number of sides.
  • $lookup monster [name] - Looks up the specified monster's stats.
  • $changeprefix [prefix] - Changes the bot's command prefix.
